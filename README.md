# 🏠 ChillDoor
부산소프트웨어마이스터고 학생들의 기숙사 생활을 더 편리하게 해주는 **기숙사 도우미 앱** 입니다.  
학생들이 기숙사에서 필요한 정보를 쉽게 확인하고, 숙장들과 사감선생님들께서 기숙사를 효율적으로 관리할 수 있도록 돕습니다.

## 커밋 컨벤션 

- `feat` : 기능 추가  
- `fix` : 버그 수정  
- `docs` : 문서 작성 및 수정
- `style` : CSS/디자인 수정
- `refactor` : 코드 개선
